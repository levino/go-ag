# Partieprotokoll (Kifu)

Ein **Kifu** (japanisch: 棋譜) ist ein Partieprotokoll, auf dem die Züge einer Go-Partie
festgehalten werden. So kann man seine Partien später nachspielen und analysieren.

Hier stehen Partieprotokolle für die gängigen Brettgrößen zum Download bereit — als
druckfertige PDF-Datei und als editierbare [Typst](https://typst.app/)-Quelldatei.
